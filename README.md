KidsMathQuiz
============

Sample
math_quiz_gen.py 1000 -o + - -r 100 -nc


usage: math_quiz_gen.py [-h] [-r RANGE] [-o [{+,-} [{+,-} ...]]] [-nc [NC]]
                        [count]

Create math quiz for kids

positional arguments:
  count                 how many quiz to generate

optional arguments:
  -h, --help            show this help message and exit
  -r RANGE, --range RANGE
                        range of numbers
  -o [{+,-} [{+,-} ...]], --operator [{+,-} [{+,-} ...]]
                        operator
  -nc [NC], --no_carry [NC]
                        not allow carry
